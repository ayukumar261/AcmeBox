The AcmeBox Transcript Flywheel

An autonomous loop that turns support-chat transcripts into LoRA fine-tuning data for the LFM2.5-8B-A1B agent — then trains, gates, and redeploys it. The improved agent produces new transcripts that feed the next turn.

Closed loop · continuous self-improvement
Pipeline generate → judge → correct → train
Return path gate & promote, closing the loop
Sources feed Conversations
Model roles:  agent being improved = LFM2.5-8B-A1B (LiquidAI, MoE)  ·  judge + teacher = strong frontier models  ·  customer / user simulator = Qwen on Together  ·  author = scenario writer.
